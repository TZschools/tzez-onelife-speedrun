# 六、教师篇


**以学科分类，排名不分先后**

**佚名：**我想说的是，二中老师里基本没有坏人，课上课下一定要尊重老师啊孩子们！只有你尊重老师，老师才有理由尊重你，才会关心你。都是快成年的人了，不要把基础的礼仪问题忘记了！

**匿名学长**：无论如何，老师不会害你，在他们这份职业中，不仅有职业操守，还有他们对你们的一腔热忱，他们的青春，还有他们对你们无私的爱与责任。无论如何，一定要对老师抱有尊重与感激，不要寒了老师们的心，先做人，再读书。

> **投稿示例**<br>
>  ✘✘✘老师<br>
>  **讲课风格：**年轻老师，很认真，课也讲的很好<br>
>  **特点：**多愁善感，温柔，超美，爱笑，很负责<br>
>  **听课建议：**跟着走就行，她说干啥你干啥<br>
>  **课后复习&学习建议：**多读课外书<br>
>  **其他：**课下超级和蔼<br>
