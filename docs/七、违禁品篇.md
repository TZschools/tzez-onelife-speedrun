# 七、违禁品篇 :id=top

## 注意事项
1. 二中查违禁品，会不定期去各厕所检查吸烟，也会趁学生上课期间去宿舍搜查违禁品。前年高三年级依靠金属探测门查过违禁品，进校时需要过门，但是效果不是很好（因排队耗时过长未完成排查便结束）。
2. 严守法律底线，步入高中后的我们即将成年，不再受《未成年人保护法》的特殊保护，一些看似无意的行为也可能触犯法律。知法懂法，对自己的行为负责，坚决对所有违法行为说不。
3. 不要引战！跟同学发生矛盾是难免的，能私下解决最好，不要拉小团体。我们来二中是来学习的，不是天天处心积虑跟别人勾心斗角的。
4. 严禁作弊，不要为了一时的侥幸，拿自己的未来和名誉开玩笑。。
5. 严禁抽烟，校园内大部分区域都会有老师进行随机检查。
6. 谨慎对待假期兼职，一方面要提高警惕，谨防刷单、虚假招工等各类诈骗陷阱；另一方面，不建议普通高中生参与兼职，若非家庭特殊情况万不得已，不要为了眼前的小额利益廉价消耗自己的时间。于你们而言，时间是最珍贵的财富，哪怕是用来合理放松，也远比做无意义的体力工作更有价值，**所谓的 “体验生活、获取社会经验”，大多只是自我安慰，反而会带来精神的疲惫与内耗。**

## 手机使用指南
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;在编者毕业时，AI大模型尚未兴起，仅仅是哔哩哔哩这一个视频软件，就已能为同学们提供密度极高的知识信息。无论是对于促进教育公平，还是对于提升个人素养而言，智能手机及其各类应用所带来的时代大潮，我们都应坦然拥抱，而非视如毒蝎。它将原本零散的时间充分利用起来，也让知识付费的成本降到了最低。但考虑到高中生自控力较弱，稍有不慎就可能陷入手机依赖的深渊，我们首先要明确：**真正能够掌控手机、而非被手机掌控的标准是什么**。而要弄清楚这一点，不妨先从自控力说起。

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;**真正的自控力，源于个人意识的觉醒**。这种觉醒，包括但不限于卧薪尝胆的坚韧、知耻而后勇的决心，或是意识到自身未来的无限可能，或是背负着改变家族命运的使命，亦或是秉持着“为中华之崛起而读书”的信念。当你完成这种意识觉醒后，自控力和内驱力会让你自发地掌控手机，而非被动地被他人约束。那么，如何才能完成个人意识的觉醒？你必须学会直面那些“万不得已”的现实，学会破釜沉舟、不破不立，主动将自己置于没有退路的境地，而不是等到高考失利的打击来临，才在逆境中被迫觉醒。比如，你可以想一想，按照如今的就业形势，若不能考上好大学、选择好专业，就很难找到理想的工作，而家人也未必能一直为你兜底；再比如，若你好面子、有傲骨，就该告诉自己：大丈夫生于天地之间，岂能郁郁久居人下。甚至，你可以在万不得已时，向所有人公开一个看似难以完成的目标——这听起来或许有些幼稚，但实则有用。破釜沉舟，切断所有退路，时刻想着失败后的尴尬与窘迫，**用自己最害怕的事情作为赌注，才能有效约束自身的懒惰**。不过，遗憾的是，大多数人的个人意识觉醒，都集中在高三初期或中期，此时已然太晚。我们更希望大家能更早觉醒，早日享受线上知识洪流带来的红利。因此，下面我们来谈谈一些具体的自控手段，而非单纯依赖个人觉醒。

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;首先强调第一点：不建议各位同学住全宿。对于大部分高中生而言，全宿生活意味着使用手机很容易走向两个极端——要么完全用不上，要么天天熬夜刷手机。当然，若实在不得不住全宿，且个人意识尚未觉醒，平时就尽量不要携带手机；周末则可以成为你提升自己的主战场，多利用手机进行学习。第二，回家后使用手机时，尽量选用带有功能限制的设备，仅开放哔哩哔哩、作业帮、AI大模型等学习类软件，或是人民日报等资讯类软件。诚然，若你执意要刷视频，没有人能够拦住你，但有约束总比无约束要好——对于那些有意愿改变自己的同学来说，他们自然不会将娱乐软件当作主要用途。

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;也就是说，个人意识的觉醒，或是主动的自我约束，是自由掌控手机的基础。接下来，我们谈谈如何用好手机、借力成长。AI和作业帮可以作为除老师之外的第三方答疑工具，帮助你快速解决学习中的难题；哔哩哔哩的网课、各类学习资料，则可以作为课后提升的补充手段。要知道，老师没有分身术，无法同时兼顾每一位同学的薄弱点，很多高一遗留的知识漏洞，可能要等到高三才能得到解决。而最了解自己短板的，永远是你自己。学会用手机辅助学习，能极大提升学习效率；更重要的是，    。

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;最后再次强调：如果你真的控制不住自己，且个人意识暂时没有觉醒，请一定要借助外界的力量约束自己使用手机。这不是不相信各位，而是我深知，未觉醒时的我们，往往很难战胜自身的惰性。
