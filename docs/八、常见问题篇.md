# 八、常见问题篇 :id=top

1. **半走读**  
半走读指的就是中午住校，而晚上回家的上学方式，军训期间原则上二中严禁半走读。想要半走读的同学，可以在缴费时选择住宿，占住一个床铺，不然开学后如果没有床位也没办法。开学后可跟老师商量半走读，办理需要家长到场，签一堆名。如果学校不允许的话，可以跟老师商量选择请一个星期的晚上宿舍住宿假。办过一次后，期末时学校会发一张统计表，统计高二半走读的学生，所以仅高一阶段麻烦一点，高二高三就好了。

2. **校车**    
分为两类，一类为每天固定送来，送回的定制公交。另一类为仅两周一次周末放假时学校定制的前往乡下及车站的车。  
学校有定制公交，每天早上送来，晚上接走，基本是给半走读准备的。有几条线路，如果没有自己的线路的话，也可申请新线路，人数足够的话就会开启。一个月120元，**具体线路就需要等待同学们的投稿了**。  
每周末放假时，学校会安排部分长途汽车，5元一次，线路很多，往各个乡镇，但是很多车往往爆满。

3. **选科、赋分是什么意思**  
高二后，从副科的六门中选择三门作为自己的高考科目，其他不再学习。但是因政策影响，也并非是随意选择三门，24年的政策是物化能分开选，但就是没法选理工科。   
分科后，由于不同科目难度不同，所以采取赋分制。赋分即为按照名次给分，前1%赋100分，1%-2%赋99分依次类推，最后1%赋21分。~而二中通过多年经验判断，决定将赋分满分定为80分。~  
具体的赋分规则，请参考[山东省人民政府办公厅关于印发山东省深化高等学校考试招生综合改革试点方案的通知](http://m.sd.gov.cn/art/2018/4/19/art_100623_22993.html)

4. **合格考是什么**  
合格考即考验学生必修的掌握程度，合格考只有合格和不合格两种，有不合格则无法领取高中毕业证。     
考试范围基本为必修内容。**高一阶段考化学、生物、历史、地理。高二阶段上学期考数学、外语、语文、政治、物理，下学期考通用技术、信息技术**。  
合格考全省掐5%的同学过不了，所以没有分数线一说。但基本不会超过20分。（外语场记得提前进考场，不然前几分钟就进不去了）注意，合格考不过的话，高考无法报考该科目，但每次合格考均有补考考场，真没过压力也别太大。但是如果遇到市级、省级优秀学生评选等活动，有未合格的将没有报考资格

5. **高考外语**  
高考中可从英语、俄语、日语、德语、法语、西班牙语中任选一门语言考试，所以**二中除英语外还开设了日语学科**（校外机构进校辅导）。从高一的暑假开始学习，以后的每个寒暑假都有开课（必上）。在校内则采取走班的方式上课，即英语课时，其他同学上英语，而日语生则前往日语教室上课。  
二中有远大、青卓两家日语机构，同学们可前往咨询，认真考量选择（不要什么哪个机构负责哪个级部忽悠了，认真分析两家的差别才是最重要的）。  
日语学习费用为2万6千元，签订保分合同，只要学生上课学习阶段没有过多的违纪行为，如果考不到 100 分，就可以按比例退款。  
近年来日语难度有所提高，但难度依旧小于英语，日语高考内容为：听力30分 + 阅读理解4篇共50分 + 完形填空15分 + 语法填空15分 + 写作两篇10分和30分。但近年来日语听说要改革，增加的一篇作文已经加了，总之难度大概率是逐年上涨的。

6. **学校可不可以打球**  
学校允许学生打乒乓球、羽毛球、篮球、排球。但原则上不允许学生在教学楼内打球，打闹（学校有排球场，但一般都被羽毛球占用）

7. **高考须知**  
24年开始外语听力考试已经改为6月统一考。在高考中，每个人主科考场为同一个，副科考场一般每场都不一样（同一个学校，但是不一定是一个教室）。高考时可以索要草稿纸，但是需要在第一张使用完成后，才能发放第二张，如果监考老师不给可以去找主考官。  
近几年新出的AI巡考大家也不要担心，AI巡考的灵敏度虽然很高，但是每一条警告都会有专人一一核查，不会出现动一下就判作弊的情况。
